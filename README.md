<p>Highly Efficient mining pool designed for Ethash based coins
<a href="https://travis-ci.org/techievee/ethash-mining-pool" rel="nofollow"><img src="https://camo.githubusercontent.com/33257f74d08940ca194114f5be6fa78471082a1d/68747470733a2f2f7472617669732d63692e6f72672f7465636869657665652f6574686173682d6d696e696e672d706f6f6c2e7376673f6272616e63683d56322e305f457468" alt="Build Status" data-canonical-src="https://travis-ci.org/techievee/ethash-mining-pool.svg?branch=V2.0_Eth" style="max-width:100%;"></a></p>
<p><a target="_blank" rel="noopener noreferrer" href="https://raw.githubusercontent.com/techievee/ethash-mining-pool/master/images/FrontEnd.png"><img src="https://raw.githubusercontent.com/techievee/ethash-mining-pool/master/images/FrontEnd.png" alt="alt text" style="max-width:100%;"></a></p>
<h3><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h3>
<p>**This pool is reegineered from sammy007 open-ethereum-pool for efficiency and for better payment algorithm. This software is functional and tested and implemented in big Mining farms. Testing and bug submissions are still welcome!</p>
<ul>
<li>Support for HTTP, SSL, Stratum, Stratum+SSL mining</li>
<li>Detailed block stats with luck percentage and full reward</li>
<li>Failover node instances: high availability built in (Any number of full nodes can be added to the configurations)</li>
<li>Modern beautiful Ember.js frontend for Individual coin statistics vs consolidated coin statistics</li>
<li>Separate stats for workers: can highlight timed-out workers so miners can perform maintenance of rigs</li>
<li>JSON-API for statistics, (Looking for contributor to build app for the pool)</li>
